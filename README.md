# 嘉豪俱乐部

一个基于 Firebase 开发的商城下单系统。

## 功能

- 用户注册
- 用户登录
- 首页
- 商品分类
- 商品详情
- 在线下单
- 二维码付款
- 上传付款截图
- 钱包
- 在线客服
- 我的订单
- 管理后台
- 商品管理
- 订单管理
- 用户管理

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6)
- Firebase Authentication
- Firebase Firestore
- Firebase Storage
